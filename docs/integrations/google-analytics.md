# Google Analytics Integration

Use Google Analytics (GA4) for tracking:
- Acquisition and attribution
- Funnel performance
- Conversion events

## Recommended Approach
- Capture UTM parameters at lead creation
- Persist attribution metadata on the lead/campaign
- Emit conversion events when leads are qualified/converted
