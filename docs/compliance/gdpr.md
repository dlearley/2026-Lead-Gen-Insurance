# GDPR Compliance

This page summarizes GDPR-related considerations for operating the Insurance Lead Generation AI Platform.

For the detailed internal compliance guide, see [GDPR_COMPLIANCE.md](../GDPR_COMPLIANCE.md).

## Key Topics

- Lawful basis and consent management
- Data minimization and retention
- Right to access / export
- Right to deletion
- Security controls and audit logging

## Operational Checklist

- [ ] Ensure consent fields are captured where required
- [ ] Provide a process for data subject requests (DSRs)
- [ ] Verify masking/redaction for logs and analytics
- [ ] Ensure vendors are covered by DPAs
