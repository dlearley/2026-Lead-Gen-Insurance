# Service Level Agreement (SLA) (Template)

This page provides a template SLA outline. Define targets based on your production SLOs.

## Suggested Sections
- Availability target (monthly)
- Definitions (downtime, maintenance windows)
- Support response times by severity
- Exclusions (planned maintenance, third-party outages)
- Credits (if applicable)
