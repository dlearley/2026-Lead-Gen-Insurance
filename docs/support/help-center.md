## Submitting a Support Request

### What to Include

For fastest resolution, include:

1. **Description**: Clear description of the issue
2. **Steps to Reproduce**: What you did before the issue
3. **Expected vs Actual**: What you expected vs what happened
4. **Screenshots**: Screenshots or screen recordings
5. **Request ID**: If you have one from error messages
6. **Browser/Device**: Browser version or device type

### Sample Support Request

```
Subject: Lead not syncing to Salesforce

Description:
Leads created in the platform are not appearing in Salesforce.
I've checked the sync status and it shows "Connected" but no leads
have synced in the last 24 hours.

Steps to Reproduce:
1. Create a new lead in the platform
2. Wait 5 minutes
3. Check Salesforce - lead doesn't appear
4. Check sync logs - no errors shown

Expected: Lead should appear in Salesforce within 5 minutes
Actual: Lead never syncs

Request ID: req_abc123456
Browser: Chrome 120
```
